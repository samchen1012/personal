# 個人簡歷範本

這是一個簡潔的個人簡歷單頁範本，包含：

- `index.html`：主頁面。
- `styles.css`：樣式表（含深色主題與列印樣式）。
- `script.js`：主題切換、平滑滾動與列印功能。

快速上手：

1. 在檔案總管中開啟 `index.html`（或用瀏覽器開啟）。
2. 編輯 `index.html` 中的文字內容以填入你的資料：名稱、經歷、專案等。
3. 若要列印為 PDF，按頁面上的「列印」按鈕或使用瀏覽器列印功能（建議選擇 A4、背景圖形勾選）。

自訂建議：

- 替換大頭照：`<img class="avatar" src="...">`
- 修改主題預設：`localStorage` 裡的 `theme`（`light` 或 `dark`）。
- 若要把表單連接實際郵件或 API，可替換 `script.js` 中的模擬送出邏輯。
